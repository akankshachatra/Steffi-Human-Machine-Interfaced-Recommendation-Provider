Steffi: Human Machine Interfaced Recommendation Provider 
In this project, we utilized Google, Locu, Weather APIs to provide useful suggestions related to weather forecasts, local restaurants etc. along with user specific needs and assessed the results obtained by interacting with the user using Python

1. Run the program
2. A question will pop up asking you How are you feeling today?
3. The answer can be 'good' , 'great' , 'awesome' or 'not that great'
4. Depending upon the answers which are given above, various other suggestions will be provided like information about local restaurants, the weather of that particular area etc.
